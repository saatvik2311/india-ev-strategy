Your Answer: "IRR is defined as the exact discount rate that makes the NPV equal to exactly zero. Because the discount rate r is in the denominator of 20 different polynomial fractions, you cannot solve for it using basic algebra. Instead of using a black-box Python library, I wrote a Bisection Search Algorithm from scratch. Here is how it works: The algorithm guesses two extremes: a 0.1% return (lo) and a 500% return (hi). It tests the midpoint. If it guesses a 50% discount rate and the NPV is still positive, it means the actual IRR is higher than 50%. It moves the lo bound to 50% and cuts the search space in half. It repeats this bisection 80 times. Because it halves the search space every single loop (O(logn) time complexity), it mathematically converges on the exact, hyper-accurate IRR in milliseconds.
